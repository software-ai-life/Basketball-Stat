# 🏀 籃球計分板 Web App

一個專為手機優化的籃球比賽計分應用，採用簡約小眾設計風格，並支援完整的數據記錄與分析功能。

## 功能特點

- ✨ **雙隊計分** - 同時記錄兩隊得分與統計數據
- 📊 **詳細統計** - 投籃命中率、抄截、籃板、助攻等完整數據
- 👤 **球員管理** - 快速識別球員，避免眼花
- 📱 **手機優化** - 針對觸控操作優化的介面
- 🎨 **優雅設計** - 米色背景、橙色強調色的精緻視覺風格
- 💾 **數據保存** - 使用 Supabase 雲端資料庫儲存比賽記錄
- 📅 **歷史記錄** - 查看每天的所有比賽記錄
- 📈 **個人統計** - 累積球員表現，含命中率排行榜

## 新功能亮點

### 🎯 投籃命中率追蹤
- 2分球/3分球分別記錄「投球」和「進球」
- 即時計算命中率百分比
- 詳細的投籃數據統計 (進球數/出手數)

### 💾 比賽數據保存
- 每場比賽完成後可保存至雲端
- 自動記錄比賽時間與日期
- 不會遺失任何比賽數據

### 📅 歷史記錄頁面
- 按日期分組顯示所有比賽
- 標示勝負結果
- 快速查看比分

### 📊 個人統計分析
- 球員總得分排行榜（🥇🥈🥉）
- 投籃命中率統計
- 場均得分
- 綜合數據（籃板、助攻、抄截、阻攻）
- 失誤統計

## 開始使用

### 1. 安裝相依套件

```bash
npm install
```

### 2. 設定 Supabase（數據保存功能）

請參考 [SUPABASE_SETUP.md](./SUPABASE_SETUP.md) 完成資料庫設定。

簡要步驟：
1. 註冊 Supabase 帳號
2. 建立新專案
3. 執行 SQL 腳本建立資料表
4. 複製專案 URL 和 API Key
5. 建立 `.env` 檔案並填入金鑰：

```env
VITE_SUPABASE_URL=你的專案URL
VITE_SUPABASE_ANON_KEY=你的anon_key
VITE_ADMIN_PASSWORD=你的管理員密碼
```

⚠️ **重要**：`VITE_ADMIN_PASSWORD` 用於刪除比賽等敏感操作，請設定一個安全的密碼。

### 3. 執行開發伺服器

```bash
npm run dev
```

應用將在 http://localhost:3000 執行

### 4. 建構生產版本

```bash
npm run build
```

### 5. 部署到 Zeabur（推薦）

```bash
git push origin main
```

詳細部署步驟請參考 [DEPLOYMENT.md](./DEPLOYMENT.md)

- ⚡ 1-2 分鐘完成部署
- 🌏 亞洲節點，台灣訪問速度快
- 🚀 自動 CI/CD，推送即部署

## 使用流程

### 比賽前設定
1. 輸入主隊名稱和球員名單
2. 輸入客隊名稱（客隊只記錄總分）
3. 點擊「開始比賽」

### 比賽中記錄
**主隊記錄（詳細數據）：**
- 選擇球員
- 投籃：點擊「未進」或「✓ 進球」
- 其他數據：點擊對應按鈕（抄截、籃板等）

**客隊記錄（快速計分）：**
- 直接點擊 +1、+2、+3 加分按鈕

### 比賽後保存
- 點擊「💾 保存比賽」儲存至雲端
- 或點擊「放棄比賽」重新開始

### 查看數據
- **📅 歷史記錄**：查看所有比賽記錄
- **📊 個人統計**：查看球員累積表現與排行

## 技術堆疊

- **前端框架**：React 18
- **樣式**：Tailwind CSS
- **建構工具**：Vite
- **資料庫**：Supabase（PostgreSQL）
- **響應式設計**：手機優先

## 資料結構

### games 資料表
- 比賽日期、隊名、比分
- 自動記錄建立時間

### player_stats 資料表
- 球員姓名、所屬球隊
- 投籃數據（2分/3分 進球數與出手數）
- 其他數據（抄截、籃板、助攻、阻攻、失誤）
